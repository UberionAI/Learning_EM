### Требования
- Go 1.21+
- Docker / Docker Compose
- Kafka доступна по адресу localhost:9092
- image: confluentinc/cp-kafka:7.6.0
- image: confluentinc/cp-zookeeper:7.6.0

### Запуск Kafka

```bash
docker compose up -d
```
- Задание 1 в корне проекта
- Задания 2 - 10 находятся в директориях с соответствующими цифрами (2-10)

После запуска Kafka можно запускать задания.

### Запуск заданий (из директорий с конкретными заданиями):
```go
go run main.go
```

1. Производитель-Потребитель
2. Асинхронный производитель
3. Многопоточный потребитель
4. Присоединение к группе
5. Партиции
6. Консольный потребитель
7. Публикация сообщений с ключами
8. Потребление сообщений с ключами
9. Подтверждение сообщений
10. Автоматический коммит

### Остановка Kafka:
```bash
docker compose down
```

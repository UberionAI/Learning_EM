OpenVPN сервер развернули на WSL Debian12, клиент - OpenVPN client для Windows10:
root@WIN-EBPAHPCV4PE:~# sudo apt install openvpn easy-rsa
генерируем все необходимые сертификаты и ключи
запускаем OpenVPN server в test mode (отключив tls-auth для упрощения запуска:
cd ~/easy-rsa
sudo openvpn \
    --server x.x.x.x 255.255.255.0 \
    --proto udp \
    --port 1194 \
    --dev tun \
    --ca pki/ca.crt \
    --cert pki/issued/server.crt \
    --key pki/private/server.key \
    --dh pki/dh.pem \
    --cipher AES-256-CBC \
    --auth SHA256 \
    --verb 3
